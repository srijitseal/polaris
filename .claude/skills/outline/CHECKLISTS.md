# Section Checklists for Paper Outlining

Use these checklists to verify outline quality before moving to prose.

---

## Abstract Checklist

### Structure

- [ ] Has opportunity/context sentence? (why this matters broadly)
- [ ] States the gap clearly? (what's missing)
- [ ] Uses "Here, we..." transition?
- [ ] Includes specific results with numbers?
- [ ] Ends with broader significance?

### Quality

- [ ] Gap and results fit like lock and key?
- [ ] Non-expert could understand the main point?
- [ ] No jargon without definition?
- [ ] Total length appropriate for target journal?

### Common Issues

- Results mentioned before gap established
- Too vague (no specific numbers)
- Missing broader significance
- Gap too narrow (only specialists care)

---

## Introduction Checklist

### Structure

- [ ] Opens with broad field context? (Para 1)
- [ ] Each paragraph narrows the gap progressively?
- [ ] Final paragraph summarizes approach? (Para 4)
- [ ] "Here, we..." appears only in final paragraph?
- [ ] Does NOT include detailed results?

### Quality

- [ ] Reader can identify the specific gap being filled?
- [ ] Gap seems important and worth filling?
- [ ] Clues/prior work support the approach?
- [ ] Not a broad literature review?

### Common Issues

- Results appear in introduction
- Gap not clearly stated
- Missing "Here we..." transition
- Literature review without gap focus
- Multiple gaps introduced (violates Rule 1)

---

## Results Checklist

### Structure

- [ ] First paragraph summarizes overall approach?
- [ ] Subsection titles are declarative statements?
- [ ] Each paragraph answers a question?
- [ ] Data includes specific numbers and statistics?
- [ ] Final sentence of each paragraph interprets the finding?

### Quality

- [ ] Logical progression of evidence?
- [ ] Each subsection title could stand alone as a finding?
- [ ] Figures support the declarative titles?
- [ ] No chronological narrative of experiments?

### Calibration

- [ ] Modest or negative results framed proportionally, not spun as positives?
- [ ] Expected results presented as confirmatory baselines, not discoveries?

### Common Issues

- Subsection titles describe methods, not findings
- Data presented without interpretation
- Chronological "then we did X" structure
- Missing statistics for key claims
- Jumping between unrelated topics (zig-zag)

---

## Discussion Checklist

### Structure

- [ ] Opens with findings summary? (Para 1)
- [ ] Limitations acknowledged with context?
- [ ] Each limitation has: weakness + context + future?
- [ ] Connects back to introduction gap?
- [ ] Ends with broader impact/significance?

### Quality

- [ ] Not just repeating Results?
- [ ] Limitations feel honest, not defensive?
- [ ] Future directions are specific and actionable?
- [ ] Broader significance is credible?

### Calibration

- [ ] Limitations presented as real costs, not reframed as "opportunities for improvement"?

### Common Issues

- Repeats Results without adding interpretation
- Limitations without context (sounds apologetic)
- Doesn't connect back to the gap
- Missing broader significance
- New results introduced in Discussion

---

## Figure/Table Checklist

### Titles

- [ ] Title states the conclusion (declarative)?
- [ ] Title is bold or clearly distinguished?
- [ ] Could understand the point from title alone?

### Legends

- [ ] Methods sufficient to understand the figure?
- [ ] Panel descriptions match panel labels?
- [ ] Statistical details included where needed?

### References

- [ ] Figures referenced indirectly: "X is Y (Figure N)"?
- [ ] Not "Figure N shows X is Y"?

---

## Overall Outline Checklist

### One Central Contribution (Rule 1)

- [ ] Can state the paper's contribution in one sentence?
- [ ] Title captures this contribution?
- [ ] Every section supports this contribution?

### C-C-C at All Scales (Rule 3)

- [ ] Each paragraph has context, content, conclusion?
- [ ] Paper has intro (context), results (content), discussion (conclusion)?

### No Zig-Zag (Rule 4)

- [ ] Each topic appears in only one location?
- [ ] Related ideas use parallel structure?
- [ ] Smooth flow between paragraphs?

### First-Sentence Test

- [ ] Reading only first sentences tells the complete story?
- [ ] First sentences are declarative, not setup?

---

## Quick Verification

After completing an outline, answer these questions:

1. **Can you describe the entire paper in 2 minutes?** If not, the message isn't clear enough.

2. **Does the gap (Introduction) match the results (Results)?** They should fit like lock and key.

3. **Does the Discussion connect back to the Introduction?** The circle should close.

4. **Are all subsection titles conclusions?** If they describe methods, revise.

5. **Would a non-expert understand the Abstract?** If not, simplify.
